# ☕ Coffee Landing Page

Página institucional focada em conversão, com design moderno, limpo e totalmente responsivo.

### 🛠 Tecnologias e Conceitos:
- **Responsive Design:** Mobile-first design usando Media Queries.
- **Design Visual:** Foco em tipografia e hierarquia visual (Background em Design Gráfico).
- **CSS Moderno:** Uso de Flexbox e Grid para o layout.
